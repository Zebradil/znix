## Code research and editing

Use Serena tools for code research and editing tasks if Serena MCP is available.
Check which code introspection and editing tools are available and use them appropriately to reduce context usage.

## GitHub Interactions

Always use the `gh` CLI tool when interacting with GitHub (creating PRs, issues, checking status, etc.) rather than using the API directly or other methods.

## Tools

- use `fd` instead of `find`
- use `rg` instead of `grep`
